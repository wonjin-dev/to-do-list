## _Get Started_

***on local***
1. 터미널에 명령어 `npm i || yarn` 입력하여 프로젝트를 설치합니다<br>
2. 설치 후에 명령어 `npm start || yarn start` 를 터미널에 입력하여 프로젝트를 실행하실 수 있습니다<br>
